# 🏥 AI Patient Context Dashboard

An AI-powered healthcare dashboard built to manage patient records, generate AI summaries, and assist clinicians with patient context management.

## 📌 Features

- Dashboard with real-time patient statistics
- Patient Management (CRUD)
- Create Patient
- Edit Patient
- Delete Patient
- Patient Details Page
- AI Workspace
- AI Summary Generation
- Agent Workspace
- Draft Management
- Search Patients
- Responsive Design
- Modern UI
- Settings Page
